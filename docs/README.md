## VigilantEye Documentation (PDF)

This folder contains the script that generates a PDF explaining:
- The **CNN (MultiHeadCNN)** architecture used for GIF malware detection
- The **GIF embedding/steganography methods** used in the demo

### Generate PDF

From the repo root:

```bash
pip install fpdf2
python docs/generate_vigilanteye_pdf.py
```

Output:
- `docs/VigilantEye_Documentation.pdf`
